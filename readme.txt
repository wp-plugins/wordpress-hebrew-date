Convert dates in wordpress to Hebrew dates.

== Description ==

The plugin preview Hebrew date in date of post and date of comments.

== Installation ==

1. Upload `wp-hebdate` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

3. (Optional) if your theme call to date with `the_time()` function replace it at `the_date()`
